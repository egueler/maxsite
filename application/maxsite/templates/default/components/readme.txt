Компоненты шаблона.

Каждый компонент в отдельном каталоге.

Основной обязательный файл - компонент.php - имя файла совпадает с каталогом, либо это index.php.
Например:
	menu/menu.php
	footer/index.php
	
Необязательные файлы:
	Стили - style.css (автоподключение в секцию HEAD — лучше не использовать)
	Js — компонент.js (автоподкючение в секцию HEAD — лучше не использовать)
	Опции - options.ini и options.php
	Файл _head.php подключается в секции HEAD
	Файл _init.php подключается при инициализации MaxSite CMS
	
Если каталог компонента начинается с _ или -, то компонент игнорируется в админ-панели.

Ручное подключение компонента (на примере _menu):
	if ($fn = mso_fe('components/_menu/_menu.php')) require($fn);
